# PizzaApp

Демо-пример клиент-серверного приложения с кешированием полученных с сервера данных в локальной базе данных.

## Технологии

- Clean Architecture
- MVVM
- Hilt
- Kotlin coroutines + Flow
- SQLite, Room
- Glide
- XML
- Retrofit
- Jetpack navigation
